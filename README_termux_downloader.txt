===============================
📥 Termux Media Downloader Setup Guide
📜 تعليمات تثبيت وتشغيل أداة التحميل في Termux
===============================

🔸 English Instructions
-------------------------------
1. Open the Termux application.
2. Install the required packages:
   pkg update && pkg upgrade -y
   pkg install python ffmpeg -y
   pip install yt-dlp spotdl mutagen rich
3. Run the script:
   python termux_downloader.py
